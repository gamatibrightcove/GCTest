How to enable Time Tracking for additional objects?
To enable an object with WorkIt! follow the steps below:

Step 1: Go to Setup | Create | Objects and click on the Timing object


Step 2: In the Custom Fields & Relationships section click the New button


Step 3: Create a Lookup Relationship field and relate it to the Object that you want to enable with WorkIt! (you can add a description and customize visibility options if desired)

Step 4: If the object that you are enabling Time Tracking for is a standard Salesforce object, go to the object's fields page (for example Setup | Customize | Accounts | Fields), for a custom object go to Setup | Create | Objects | OBJECTNAME


Step 5: In the Custom Fields & Relationships section, click the New button and create a Checkbox field named 'Dont_Time' (the name must match exactly), it is not necessary to add this field to any page layouts


Step 6: Go to the WorkIt! App Setup page (Setup | View Installed Packages | WorkIt! | Configure) and verify that the object appears in the Enabled Objects list
